Redis 职责划分

session:{session_id}  → { userId, access_token, id_token }
BFF 创建，SSO /logout 时跨服务清除。TTL 24h，续期刷新。
auth_code:{code}  → { client_id, user_id, scopes, redirect_uri }
SSO 签发后写入，/token 消费后删除。TTL 5 min。
nonce:{nonce}  → { wallet_address, createdAt }
Web3 签名用，验签后删除。TTL 5 min，防重放。
rate_limit:{ip}:{action}  → counter
登录/换 token 频率限制，滑动窗口。TTL 按窗口。
